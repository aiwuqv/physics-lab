氢原子能级知识

1. 氢原子能级公式

玻尔理论给出氢原子能级公式：
En = -13.6/n² eV

其中 n = 1, 2, 3... 为主量子数

n=1 为基态，n>1 为激发态

2. 主要光谱线系

莱曼系：n→1 (紫外区)

巴尔默系：n→2 (可见光区)

帕邢系：n→3 (红外区)

布拉开系：n→4 (红外区)

蒲芬德系：n→5 (红外区)

3. 巴尔默公式

描述氢原子可见光谱：
1/λ = R(1/2² - 1/n²)

R = 1.097×10⁷ m⁻¹ (里德伯常数)
n = 3, 4, 5... 对应不同谱线

4. 电子跃迁规律

吸收过程：电子从低能级跃迁到高能级，需要吸收光子

发射过程：电子从高能级跃迁到低能级，会发射光子

光子能量：E = hν = hc/λ = |Eᵢ - Ef|

5. 重要物理常数

基态能量：E₁ = -13.6 eV

第一激发态：E₂ = -3.4 eV

里德伯常数：R = 1.097×10⁷ m⁻¹

普朗克常数：h = 6.626×10⁻³⁴ J·s

光速：c = 3.00×10⁸ m/s

6. 实际应用

氢原子光谱在天体物理学中有重要应用，可用于测定恒星成分、温度、运动速度等性质。

激光技术中也利用了原子能级跃迁原理。

教学说明

本演示展示氢原子的能级结构和电子跃迁现象：

  • 垂直轴表示能量，上方为高能级，下方为低能级
  • 水平线代表不同的能级，数字表示主量子数n
  • 点击能级可将电子移动到该能级
  • 使用跃迁控制可观察电子在不同能级间的跳跃
  • 观察光子发射/吸收过程以及对应的光谱线
