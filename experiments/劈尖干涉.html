劈尖干涉实验
劈尖由两块玻璃板左侧接触、右侧垫高形成三角形空气层。平行光垂直入射，在上下表面反射形成两束光，在竖直方向上发生干涉。
光程差公式：δ = 2d + λ/2
条纹间距公式：Δx = λ/(2θ)
两表面竖直反射光线：
上表面反射光无半波损失，竖直向上反射。
下表面反射光有半波损失(λ/2)，竖直向上反射。
两束光在竖直方向叠加形成干涉条纹。
干涉条件：
明纹：2d + λ/2 = kλ (k=1,2,3...)
暗纹：2d + λ/2 = (2k+1)λ/2 (k=0,1,2...)
(d为空气层厚度，θ为劈尖角度)

实验关键点

  • 左侧接触(d=0)，右侧垫高形成空气劈尖
  • 平行单色光垂直入射
  • 上下表面反射光均为竖直方向
  • 上表面反射无半波损失，下表面有
  • 条纹间距 Δx = λ/(2θ)
  • 角度增大→条纹变密，波长增大→条纹变疏
