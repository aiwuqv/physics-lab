核反应堆主要装置介绍：

  • 堆芯（反应区）：核燃料元件存放区域，链式反应在此进行

  • 冷却剂：水或液态金属钠，带走核裂变产生的热量
  • 防护层：厚水泥层，阻挡放射线泄漏，保障安全

核反应堆原理与数据

核反应堆工作原理

核反应堆是通过可控的核裂变链式反应产生能量的装置，其核心过程如下：

  1. 中子轰击：中子撞击铀-235原子核
  2. 核裂变：铀-235原子核分裂成两个较轻的原子核
  3. 释放中子：每次裂变释放2-3个新中子
  4. 链式反应：新中子引发更多裂变，形成持续反应
  5. 能量转换：裂变产生的能量转化为热能

通过控制棒调节中子数量，实现对反应速率的精确控制。

核反应堆工作原理

核反应堆是通过可控的核裂变链式反应产生能量的装置。其核心组成部分包括：

  • 核燃料：含有易裂变同位素（如铀-235、钚-239）的材料
  • 慢化剂：降低快中子速度，提高裂变概率（如轻水、重水、石墨）
  • 控制棒：吸收中子，调节反应速率（如硼、镉、铪）
  • 冷却剂：带走热量，防止堆芯过热（如轻水、重水、液态金属）

教学提示：尝试调节控制棒深度和冷却剂流速，观察对反应速率和安全等级的影响。

核裂变反应方程式

当一个中子撞击铀-235原子核时，会发生核裂变：

n + 235U → 236U* → 93Rb + 141Cs + 2-3n + 200MeV

或者：

n + 235U → 95Mo + 139La + 2-3n + 200MeV

每次裂变释放2-3个中子和约200MeV的巨大能量。

链式反应控制

通过调节控制棒插入深度来控制反应速率：

  • 控制棒深入：吸收更多中子 → 反应减缓 → 功率下降
  • 控制棒提升：吸收中子减少 → 反应加快 → 功率上升
  • 完全插入：吸收几乎所有中子 → 紧急停堆

理想情况下，每个裂变中子引发一个新裂变，使反应持续稳定进行（k=1）。
