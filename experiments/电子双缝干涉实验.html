电子双缝干涉实验模拟

探索电子的波动性质与干涉现象

双缝干涉理论基础

德布罗意假说指出，实物粒子也具有波动性，其波长由德布罗意公式给出：

λ = h/p = h/(mv)

其中，λ 是物质波波长，h 是普朗克常数，p 是粒子动量，m 是粒子质量，v 是粒子速度。

当电子束通过双缝时，会发生干涉现象，这证实了电子的波动性。双缝干涉会在屏幕上形成明暗相间的直线条纹，这是由于从两个狭缝发出的电子波相互叠加造成的。当两列波同相位时，产生相长干涉形成亮条纹；当两列波反相位时，产生相消干涉形成暗条纹。

双缝干涉条纹间距由公式决定：Δx = Lλ/d，其中 L 是双缝到屏的距离，λ 是电子波长，d 是双缝间距。

双缝干涉原理

双缝干涉的条纹间距由以下公式确定：

Δx = Lλ/d

其中，Δx 是相邻亮纹（或暗纹）之间的距离，L 是双缝到屏幕的距离，λ 是电子的德布罗意波长，d 是两缝之间的距离。

当屏上某点到两缝的光程差是波长的整数倍时，出现亮纹（相长干涉）；当光程差是半波长的奇数倍时，出现暗纹（相消干涉）：

亮纹条件: Δ = nλ (n = 0, ±1, ±2, ...)
暗纹条件: Δ = (n + 1/2)λ (n = 0, ±1, ±2, ...)

这意味着，当电子的波长 λ 越长（即加速电压越低）或双缝间距 d 越小时，条纹间距越大，干涉现象越明显。
