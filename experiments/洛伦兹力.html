洛伦兹力学习检测

完成以下5个问题，测试你对洛伦兹力的理解

1. 洛伦兹力的计算公式是：

F = qE
F = qvBsinθ
F = ma
F = Gm₁m₂/r²

解析

洛伦兹力的计算公式为 F = qvBsinθ，其中 q 是电荷量，v 是电荷速度，B 是磁感应强度，θ 是速度与磁场方向的夹角。

2. 根据左手定则，当正电荷垂直进入磁场时，洛伦兹力方向：

与速度方向相同
与速度方向相反
垂直于速度和磁场所确定的平面
与磁场方向相同

解析

根据左手定则，洛伦兹力方向垂直于速度和磁场所确定的平面。对于正电荷，四指指向速度方向，磁感线垂直穿入掌心，拇指方向即为洛伦兹力方向。

3. 当电荷在磁场中静止时，洛伦兹力大小为：

最大值
0
与电荷量成正比
与磁感应强度成正比

解析

根据公式 F = qvBsinθ，当电荷静止时 v=0，因此洛伦兹力为0。洛伦兹力是磁场对运动电荷的作用力，静止电荷不受洛伦兹力。

4. 当电荷运动方向与磁场方向平行时，洛伦兹力大小为：

最大值
0
与速度大小无关
与电荷量无关

解析

根据公式 F = qvBsinθ，当电荷运动方向与磁场方向平行时，θ=0°或180°，sinθ=0，因此洛伦兹力为0。

5. 负电荷在磁场中运动时，洛伦兹力方向：

与正电荷相同
与正电荷相反
始终与速度方向相同
始终与磁场方向相同

解析

负电荷在磁场中运动时，洛伦兹力方向与正电荷相反。对于负电荷，四指应指向速度的反方向，或使用右手定则判断。
