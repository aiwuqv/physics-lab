实际应用

圆孔夫琅禾费衍射

当平行光通过圆孔时，由于光的波动性，会发生衍射现象。在夫琅禾费衍射（远场衍射）条件下，在远处的观察屏上会形成明暗相间的同心圆环，中央是一个明亮的圆斑，称为艾里斑。

I(θ) = I₀ [2J₁(x)/x]²，其中 x = (πD sinθ)/λ

其中：

  • I(θ) - 角度θ处的衍射光强
  • I₀ - 中央极大光强（θ=0处的光强）
  • J₁(x) - 一阶第一类贝塞尔函数
  • D - 圆孔直径
  • λ - 光波长
  • θ - 衍射角（与光轴的夹角）

关键特征

艾里斑：中央亮斑，包含约84%的衍射光能量。

第一暗环：出现在 x = 1.22π 处，对应 sinθ = 1.22λ/D。

次级亮环：亮度迅速衰减，第一亮环强度仅为中央亮斑的1.75%。

公式推导要点

圆孔夫琅禾费衍射的强度分布公式推导基于惠更斯-菲涅耳原理：

  1. 将圆孔面分割为无数个面元，每个面元作为次波源
  2. 计算所有次波在观察点的叠加（积分）
  3. 由于圆对称性，积分可简化为贝塞尔函数形式
  4. 最终得到归一化强度公式 I/I₀ = [2J₁(x)/x]²
第一暗环条件：J₁(x) = 0 的第一个非零根为 x = 3.8317 ≈ 1.22π

由此得到艾里斑的角半径公式：

θ₀ = arcsin(1.22λ/D) ≈ 1.22λ/D (当θ很小时)

这就是著名的瑞利判据：两个点光源恰好能分辨时，一个点光源的艾里斑中心落在另一个点光源的艾里斑第一暗环上。

实际应用

光学仪器分辨率

任何光学仪器（望远镜、显微镜、相机等）的分辨率都受到圆孔衍射的限制。根据瑞利判据，最小分辨角为：

δθ = 1.22 λ / D

要提高分辨率，可以：增大孔径D 或 使用更短波长λ。

应用实例

  • 天文望远镜：大型望远镜（如哈勃望远镜D=2.4m）具有高分辨率，能分辨遥远星体的细节
  • 显微镜：使用短波长的紫外光或电子束（电子显微镜）来提高分辨率
  • 相机镜头：大光圈镜头在弱光下表现更好，但小光圈时衍射效应更明显
  • 激光束：激光通过小孔后的衍射图案用于测量光束质量和模式

思考与练习

问题1：根据模拟结果，分析圆孔直径对艾里斑大小和衍射图案的影响规律。

圆孔直径D越小，艾里斑角半径θ₀=1.22λ/D越大，衍射现象越明显，中央亮斑越大，衍射环间距也越大。反之，D越大，艾里斑越小，衍射图案更集中，分辨率越高。

问题2：计算波长为500nm的光通过直径2mm的圆孔后，艾里斑的角半径和在1m远处的线半径。

已知：λ=500nm=5×10⁻⁷m，D=2mm=2×10⁻³m，L=1m

艾里斑角半径：θ₀ = 1.22λ/D = 1.22×5×10⁻⁷/(2×10⁻³) = 3.05×10⁻⁴ rad

艾里斑线半径：r₀ = L·θ₀ = 1×3.05×10⁻⁴ = 3.05×10⁻⁴ m = 0.305 mm

问题3：解释为什么电子显微镜比光学显微镜有更高的分辨率？

根据瑞利判据δθ=1.22λ/D，分辨率与波长λ成反比。光学显微镜使用可见光（λ≈400-700nm），而电子显微镜使用电子束，其德布罗意波长比可见光短得多（典型值0.001-0.01nm），因此分辨率大大提高。
