📘 薄膜干涉 · 完整知识点（授课专用）

1. 完整光程差公式（含折射角）
Δ = 2 n d cosθ₂ + λ/2
• n：薄膜折射率
• d：薄膜厚度
• θ₂：薄膜内折射角
• λ/2：半波损失
2. 折射定律（斯涅尔定律）
sinθ = n sinθ₂
3. 亮纹（加强）条件：
2nd cosθ₂ + λ/2 = kλ (k=1,2,3...)

4. 暗纹（减弱）条件：
2nd cosθ₂ + λ/2 = (k+0.5)λ (k=0,1,2...)

📌 核心结论

✔ 入射角越大 → 折射角越大 → cosθ₂ 越小 → 光程差越小
✔ 厚度均匀 → 等倾干涉
✔ 厚度变化 → 等厚干涉（劈尖、牛顿环）
✔ 白光 → 彩色条纹；单色光 → 明暗条纹
